Пояснительная записка  
Финансовый сайт для отслеживания курсов акций, валют и криптовалют  

1. Введение  
Цель проекта: Разработка веб-приложения на Python (Flask) для отслеживания курсов акций, валют и криптовалют с возможностью добавления и удаления активов, а также расчета прибыли/убытков.  

Актуальность: Пользователи финансовых рынков нуждаются в удобном инструменте для мониторинга своих инвестиций в режиме реального времени. Данное приложение позволит им быстро оценивать текущее состояние портфеля.  

2. Функциональные требования  
Приложение должно предоставлять следующие возможности:  
- Отображение курсов в реальном времени (или с задержкой) для:  
  - Акций (например, через API Yahoo Finance, Alpha Vantage)  
  - Валют (например, через API Open Exchange Rates)  
  - Криптовалют (например, через CoinGecko или Binance API)  
- Добавление и удаление активов в портфель.  
- Расчет прибыли/убытков по каждому активу на основе цены покупки.  
- Визуализация данных (изменения цен).  
- Аутентификация пользователей (регистрация, вход, личный кабинет).  

3. Технологический стек  
- Backend: Python + Flask  
- Frontend: HTML, CSS, Bootstrap
- База данных: SQLite (для хранения данных пользователей и их активов)  
- Внешние API:  
  - Yahoo Finance / Alpha Vantage (акции)  
  - Open Exchange Rates / Forex API (валюты)  
  - CoinGecko / Binance API (криптовалюты)
